ОРИС

Создать многостраничный сайт:

  1. Не менее 4 разных страниц (почти не отличающиеся друг от друга страницы не считаются)
  2. На всех страницах должна быть навигационная панель по всем страницам сайта
  3. Должны быть использованы все теги HTML, которые мы использовали на паре, плюс еще как минимум 3 новых, которые вы найдете сами
  4. На всех страницах должен быть выдержан единый стиль оформления CSS
  5. В CSS должны быть использованы все 3 способа доступа к элементам страниц (по тегу, по классу, по айди)

Установить PostgreSQL
